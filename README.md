# Bessel-2p-analysis-pipeline

Welcome to the **Bessel-2p-analysis-pipeline**, a MATLAB-based toolkit designed for analyzing calcium imaging data collected using either Bessel beam microscopy or traditional 2-photon microscopy. This pipeline guides users through preprocessing TIFF stacks, manually selecting ROIs, computing Î”F/F (delta fluorescence over baseline), aligning imaging data with behavioral triggers, post-processing, and generating publication-ready data outputs.
